Dokumentation för Header samt HeaderMenu

Översikt
<vgr-header> är en komponent som skall användas i huvudet på sidorna och ligga där fast när användaren navigerar i systemet.
Färgen kan sättas beroende på vilket system som är aktivt. I portalen beror färgen på om användaren har ett eller flera system. Om bara ett system finns, är det den färgen som gäller. Headern innehåller en transparent bild som gör att man bara behöver sätta headerns bakgrundsfärg genom rätt klassnamn på headern eller på ett föräldraelement. I det "neutrala" läget skall inte den transparenta bilden synas. Headern kan visas i olika utföranden, man kan ha den med eller utan användarinformation samt med eller utan meny och olika kombinationer av dessa. I exemplet kan man se de olika vyerna.

På headern kan man även lägga till en kontextmeny ( <vgr-header-menu>). Den menyn kan endast bo i headern.

API - header
Namn Beskrivning Exempel
systemText : string Text för att påvisa vilken miljö man befinner sig i. Ska inte visas i produktion. Upp till immplementatören. [systemtext]="'development'"
hideSwosh : boolean Hanterar bakgrundsbild som täcker headern. [hideSwosh]="true"
logoClass Fält för att skriva över klass för logo. [logoClass]="'custom-logo'"

API - header-menu
Namn Beskrivning Exempel
userName : string Sätter namnet på texten för den som är inloggad. Intialerna för cirkeln är default först bokstaven i första och sista namnet. Exemepel: "Göte Borg" får initialerna GB, och "Göte Borg Stad" får GS [userName]="signedInUser"
initials : string Skriver över initialerna med egna initialer [initials]="GB"
textColor : string Färgen på texten i ringen. Accepterar html-kompatibla färgväden (HEX eller färgnamn) [textColor]="white"
circleColor : string Färgen på cirkeln. Accepterar html-kompatibla färgväden (HEX eller färgnamn) [circleColor]="'#f9dede'"
