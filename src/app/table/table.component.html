<vgr-table> består av <vgr-table-header> och <vgr-table-row> Headern är alltid synlig och kan vid klick visa/dölja tabellens rader. Använd table när du vil ha en hierarki i dina listor. En vgr-table kan även innehålla en vgr-table i sig. Se förslag på användning under rubriken exempel.

<vgr-table>

Namn Beskrivning Exempel
expanded: boolean Anger om sektionen är expanderad eller ej [expanded]="true"
expandable: boolean (default: true) Genom att sätta [expandable]="false" på tabellen så kan kan raden inte expanderas och har ingen chevron. [expandable]="false"
expandedChanged: EventEmitter<boolean> Event som triggras när sektionen expanderas/kollapsas. (expandedChanged)="onExpandedChanged($event)"

<vgr-table-header>

Beskrivning
Behållare för vgr-table-header-column. Har inga properties.

<vgr-table-header-column>

Namn Beskrivning Exempel
width : number vgr-table är baserad på en 20 kolumns layout. Varje kolumn måste ha en width satt och summan av width ska vara 20 för vgr-table. [width]="20"
align : string Man kan sätta en alignment på kolumnerna. Defaultvärde är left. Giltiga värden är right, left och center.
Om man sätter ett felaktigt värde så sätts defaultvärdet 		[align]="'right'"

<vgr-table-row>

Beskrivning
Behållare för vgr-table-row. Har inga properties.

<vgr-table-row-column>

Namn Beskrivning Exempel
width : number vgr-table är baserad på en 20 kolumns layout. Varje kolumn måste ha en width satt och summan av width ska vara 20 för vgr-table. [width]="20"
align : string Man kan sätta en alignment på kolumnerna. Defaultvärde är left. Giltiga värden är right, left och center.
Om man sätter ett felaktigt värde så sätts defaultvärdet 		[align]="'right'"

Navigering i tabell:

  • - Tab/Shift+Tab - För att hoppa mellan raderna
  • - Enter eller Space - För att toggla expanderbara tabellrader mellan expanderat och ihopfällt läge.
  • - Implementatören behöver sätta tabindex på sina table rows för att de skall vara möjliga att nå med tab.
