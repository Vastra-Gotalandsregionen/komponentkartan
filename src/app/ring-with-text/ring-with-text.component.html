<vgr-ring-with-textn> Kan användas tillexempel för att visa upp en inloggad användares initialer. Det går att styra både text, färg och storlek.
Här är ett exampel, det går att se fler under exempel här under.

Namn Beskrivning Exempel
text : string Texten som skall stå i ringen (ett eller två tecken). Skriver man fler tecken så visas endast de två första och skriver man ingen text visas ingen ring. text="NE"
size : string Storleken på cirkeln. Värdena den accepterar är "small" (default) och "large". size="small"
textColor : string Färgen på texten i ringen. Accepterar html-kompatibla färgväden (HEX eller färgnamn) textColor="white"
circleColor : string Färgen på cirkeln. Accepterar html-kompatibla färgväden (HEX eller färgnamn) circleColor="#000000"
