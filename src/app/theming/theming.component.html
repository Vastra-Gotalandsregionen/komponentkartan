Använda tema från Komponentkartan

För att återanvända stilar och teman från komponentkartan används primärt komponentkartans CSS / SASS filer. Importera dem i index.html eller angular-cli.json.


index.html==> <link rel="stylesheet" type="text/css" href="node_modules/vgr-komponentkartan//Content/komponentkartan.css" >
angular-cli.json==> "styles": [ "styles.scss", "../node_modules/vgr-komponentkartan//Content/komponentkartan.scss" ],

Temadefinierande klasser

För att välja tema för applikationen används klasserna .theme--neutral, .theme--blue, .theme--green och .theme--red. Sätt temat så högt upp i hierarkin som möjligt (t.ex body, applikation) för att slå igenom på samtliga sidor och komponenter.

Klasser som använder tema

För att temafärgerna skall få effekt finns en rad klasser i komponentkartan som sätter den valda temafärgen på ett element. Dessa följer standarden [css-egenskap]-- [primär/sekundärfärg]. Se exempel nedan.


  • Högerkant i primärfärg .border-right-color--primary
  • Överkant i primärfärg, bakgrund i sekundärfärg .border-top-color--primary .background-color--secondary

Använda SASS från komponentkartan

Det är också möjligt att använda komponentkartans SASS-filer i applikationen. Man kan då få tillgång till standardfärger, storlekar mm som komponentkartan använder. OBS! Vid import av SASS-filer till egendefinierade SASS/CSS filer så kompileras innehållet utanför komponentkartan. Detta kan medföra sidoeffekter så som att temat åsidosätts. Användning av klasser är att föredra.
