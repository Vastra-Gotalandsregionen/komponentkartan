Under "kodexempel" finns exempel på hur input-komponenten kan användas i Angular Reactive Forms:
Reactive Forms exempel
Property Beskrivning Exempel
value: string Värde att binda till [value]="person.lastName"
showValidation: boolean Anger om valideringsfel ska visas eller ej. Default = true. [showValidation]="true"
maxlength: string Max antal tecken. Begränsar antalet möjliga tecken att mata in. Default = obegränsat. [maxlength]="5"
errorMessage: string Text som visas vid valideringsfel [errorMessage]="'Ange ett värde mellan 1-10'"
suffix: string Text som visas till höger om fältet, som t.ex en enhet (km, m, kg, kr mm) [suffix]="'kg'"
alignRight: boolean Om true, högerställ innehållet i fältet. Default är False. [alignRight]="true"
formatNumber: boolean Om värdet som visas ska formateras, dvs '.' visas som ',' och mellanslag läggs mellan 1000-tal. [formatNumber]="true"
nrOfDecimals: number Antal decimaler som det inmatade värdet ska avrundas till. [nrOfDecimals]="2"
small: boolean Gör inputfälten till en mindre storlek. [small]="true"
Property Beskrivning Exempel
blur: eventEmitter Event som triggas när användaren går ifrån inputfield. (blur)="function($event)"
focus: eventEmitter Event som triggas när användaren fokuserar på inputfield. (focus)="function($event)"
valueChanged: eventEmitter Event som triggas när inputfield uppdateras med ny text. (valueChanged)="function($event)"
