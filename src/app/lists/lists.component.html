Översikt

<vgr-list> är en lista bestående av <vgr-list-item>. En lista med header och expanderbara rader, som används till ex. listor av kort. Rubrikraden är fast positionerad och kan inte scrollas bort.


Förändras med tema: nej

API - vgr-list
Namn Beskrivning Exempel
allowMultipleExpandedItems : boolean Om flera rader skall kunna vara expanderade samtidigt, default är false. [allowMultipleExpandedItems]="true"
flexibleHeader : boolean Kan man sätta om headern skall följa med i scrollningen, default är false. [flexibleHeader]="true"
notification : Notification Sätter man om man vill visa en notifiering, t.ex. när listan ej kan laddas, default visas inget meddelande. [notification]="{{'{'}} message: 'Här är texten i notifikationen', icon: 'vgr-icon-message'{{'}'}}"

API - vgr-list-item
Namn Beskrivning Exempel
expanded : boolean Anger om raden skall vara expanderad eller inte. [expanded]="false"
notification : RowNotification Ger möjligheten att lägga en notifiering på raden. En RowNotification har en icon property (string), removeWhenDone property (boolean) och en typ.
-Icon finns angivna på sidan för ikoner
-RemoveWhenDone anger om notifieringen ska försvinna efter kollapsanimationen för notifiering är klar. Default är att notifieringen som är permanent är kvar efter att en annan notifering visats.
-Notification.Type att välja på är: NotificationType.Permanent, visas i samband med att raden minimeras - NotificationType.ShowOnCollapse eller när raden tas bort - NotificationType.ShowOnRemove 		[notification]="{{'{'}} message: 'Personen är inaktiv', icon: 'vgr-icon-message', type: notificationTypes.Permanent{{'}'}}" OBS! För att binda till enums i html, måste en rad till i .ts-filen som deklarerar enum som en property. T.ex: notificationTypes = NotificationType
expandedChanged : EventEmitter<boolean> Event som triggas när raden expanderar (expandedChanged)="onExpandedChanged($event)"
notificationChanged : EventEmitter<RowNotification> Event som triggas när notifiering ändras (notificationChanged)="onNotificationChanged($event)"
deleted : EventEmitter<any> Event som triggas när item har tagits bort (visuellt) (deleted)="onDeleted()"

API - vgr-list-column-header
Namn Beskrivning Exempel
text : string Kolumnens rubrik [text]="'Förnamn'"
sortDirection : SortDirection Anger åt vilket håll kolumnen sorteras vid initiering SortDirection.None, default - ingen sortering SortDirection.Ascending sortera stigande SortDirection.Descending sortera fallande [sortDirection]="sortDirection.Ascending" OBS! För att binda till enums i html, måste en rad till i .ts-filen som deklarerar enum som en property. T.ex: sortDirection = SortDirection
sortKey : string Nyckel som används för att identifiera kolumnen vid sorteringsevent. [sortKey]="'firstName'"
width : number vgr-list är baserad på en 20 kolumns layout. Varje kolumn måste ha en width satt och summan av width ska vara 20 för vgr-list. [width]="20"
align : string Man kan sätta en alignment på kolumnerna. Defaultvärde är left. Giltiga värden är right, left och center.
Om man sätter ett felaktigt värde så sätts defaultvärdet 		[align]="'right'"
sortChanged : EventEmitter<SortDirection> Event som triggas när sortering sker i kolumnen (sortChanged)="onSortChanged($event)"

Tillgänglighet

Navigering i listan:

- Enter och Space: När fokus är på listradens header togglas listraden expanderad / kollapsad.
- Nedåtpil: Om fokus ligger på en listrad flyttar fokus till raden nedanför. Om fokus ligger på den sista raden flyttas fokus till den första raden.
- Uppåtpil: Om fokus ligger på en listrad, flyttar fokus till raden ovanför. Om fokus ligger på första raden, flyttas fokus till den sista raden.
- Home: När fokus ligger på en listrad, flyttar fokus till första raden.
- End: När fokus ligger på en rad flyttar fokus till sista raden.
- Ctrl + Page Down: Om fokus ligger inuti en rad eller på en rad, flyttar fokus till nästa rad. Om fokus ligger i den sista raden eller i raden, flytta fokus till första raden.
- Ctrl + Page Up: Om fokus ligger inuti en rad flyttar fokus till huvudet för aktiv rad. Om fokus ligger på en rad, flyttas fokus till föregående rad. Om fokus ligger på första raden, flytta fokus till sista raden.
