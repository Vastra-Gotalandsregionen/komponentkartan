Under "kodexempel" finns exempel på hur checkbox-komponenten kan användas i Angular Reactive Forms:
Reactive Forms exempel
Namn Beskrivning Exempel
label : string Text som man vill sätta på checkboxen. Kan lämnas tom för att inte ha en ledtext [label]=" 'Inaktiverad' "
checked : boolean true/false, defaultvärde false. Sätter om checkboxen är vald eller inte [checked]=True
disabled : boolean true/false, defaultvärde false. Sätter om checkboxen är inaktiverad eller inte [disabled]=True
checkedChanged : EventEmitter<boolean> Event som triggas när man checkar i eller ur checkboxen (checkedChanged)="onCheckedChanged($event) "

<vgr-checkbox> har en intern div vars roll är satt till checkbox. Förutom rollen så används attributen: attr.aria-disabled (inaktiv/aktiv), attr.aria-checked (kryssad/ej ikryssad) och attr.aria-labelledby (pekning till interna label i <vgr-checkbox>)
