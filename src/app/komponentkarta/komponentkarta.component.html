Låsknapp

Låsknappen är en "toggle-knapp ", som kan vara låst eller upplåst. För att inaktivera sätt [disabled]=" true". För att skapa en upplåst knapp, sätt [unlocked]=" true". Knappen skickar två events, ett för låst (onLocked) och ett för upplåst (onUnlocked).
<vgr-lock-button (onLocked)="consoleLog('Locked!')" (onUnlocked)="consoleLog('Unlocked!')"></vgr-lock-button> <vgr-lock-button [unlocked]="true" (onLocked)="consoleLog('Locked!')" (onUnlocked)="consoleLog('Unlocked!')"></vgr-lock-button> <vgr-lock-button [disabled]="true" (onLocked)="consoleLog('Locked!')" (onUnlocked)="consoleLog('Unlocked!')"></vgr-lock-button>

Textknappar

Knappar som innehåller text med en minsta bredd på 87px. Vid klick triggas event click. För att inaktivera sätt [disabled]=" true". För sekundär knapp sätt [secondary]=" true". Knappen stödjer klick med tangentbord via Space och Enter, enligt WCAG Button examples Notera: Knapparna nedan loggar till konsollen vid klick.
