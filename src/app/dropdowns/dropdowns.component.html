Under "kodexempel" finns exempel på hur dropdown-komponenten kan användas i Angular Reactive Forms:
Reactive Forms exempel
Namn Beskrivning Exempel
items : DropdownItem<T>[] Samling av DropdownItem<T> som utgör valbara alternativ [items]="[item1, item2, ...]"
disabled : boolean Defaultvärde false [disabled]="true"
showAllItemText : string Värde för att sätta vilken text det skall stå på raden för att ta bort filtrering och visa alla. Defaultvärde"Visa alla" [showAllItemText]="Visa alla alternativ"
noItemSelectedLabel : string Värde för att sätta vilken text det skall stå i dropdownen då man inte valt något värde [noItemSelectedLabel]="Välj alternativ"
onSelectedChanged : EventEmitter<any> Event som triggas när man väljer ett nytt alternativ i dropdownen (selectedChanged)="onSelectedChanged($event)"
showValidation: boolean Anger om valideringsfel ska visas eller ej. Default = true. [showValidation]="true"
readonly : boolean Om dropdownen är satt som readonly så ska bara textfält med valt värde visas. Om inget värde är satt så visas ingenting. [readonly]="true/false"

<vgr-dropdown> har en intern div vars roll är satt till combobox. Förutom rollen så används attributen: attr.aria-disabled (inaktiv/aktiv) och attr.aria-readonly (skrivskyddad/ej skrivskyddad)
