Namn Beskrivning Exempel
selectedDate: Date Värdet på kontrollen. Gör det möjligt att programmatiskt styra värdet när form control inte används. [value]="someDate"
minZoom: string Sätter vilken typ av tidsenhet som kan väljas. För dag, använd "d", "day" eller "days". För månad, använd "m", "month" eller "months". För år, använd "y", "year" eller "years". Förvalt värde är dag. [minZoom]="'m'"
minDate: Date Anger minsta valbara datum. Om inget anges finns ingen begränsing. [minDate]="someDate"
maxDate: Date Anger största valbara datum. Om inget anges finns ingen begränsing. [maxDate]="someDate"
allowText: boolean Anger om textinmatning tillåts. Förvalt värde är true. [allowText]="false"
readonly: boolean Gör att textfält med valt värde visas. Förvalt värde är false. [readonly]="true"
disabled: boolean Förhindrar interaktion. Förvalt värde är false. [disabled]="true"
transparent: boolean Sätter om checkboxen skall ha en transparent bakgrund eller vara vit. Förvalt värde är false. [transparent]="true"
showValidation: boolean Anger om valideringsfel ska visas eller ej. Förvalt värde är false. [showValidation]="true"
errorMessage: any Meddelande som skall visas vid valideringsfel. Nyckel för meddelandet skall vara valideringens namn. Valideringen required har redan det förvalda värdet "Obligatoriskt". Detta kan ändras. [errorMessage]="{ required: 'Du måste välja ett värde.' }"
labelId: string ID för det element som är rubrik. Används av skärmläsare. [labelId]="'someLabelId'"
selectedDateChanged: EventEmitter<Date> Event som triggas när val ändras. Värdet är det nya valet. (selectedDateChanged)="doSomething($event)"
focus() Metod man kan anropa för att programmatiskt kan sätta fokus på komponenten. this.myDropdown.focus()
