<vgr-skeleton-loader> används för att visa att en åtgärd pågår. Man kan sätta olika former med width och height attribut och det finns även möjlighet att sätta en cirkulär form med cirle attributet. För att loadern ska fungera så behöver man även sätta active attributet.

Namn Beskrivning Exempel
active : boolean Anger om loader är igång eller ej. Minsta tid som loader visas är 1 sekund. Inaktivering kan därför fördröjas. [active]="true"
width : string Sätt bredden på skeleton-loader width="130px"
height : string Sätt höjden på skeleton-loader height="130px"
circle : boolean Visar loadern i ett cirkulärt format [circle]="true"
