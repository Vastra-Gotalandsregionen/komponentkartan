Dokumentation för tillgängliga Pipes i komponentkartan

Det finns ett visst antal olika pipes som man via komponentkartan kan använda sig av, utöver de inbyggda som redan finns i Angular.


Namn Beskrivning Exempel
hsaidPipe : string Pipe som tar in ett HSA-id (SEZZZZZZZZZZ-E0XXXXXXXXXX60) och returnerar sista delen av HSA-id:t (E0XXXXXXXXXX60) 'SEZZZ-E0XXX60' | hsaidPipe
postnummer : string Pipe som tar in en sträng (5 siffror) och returnerar ett svenskt postnummer format (ex. XXX XX) '47700' | postnummer
personnummer : string Pipe som tar in en sträng (10 eller 12 siffror) och returnerar ett svenskt personnummer format (ex. YYYYMMDD-XXXX) '191212121212' | personnummer
komponentkartanCurrency : number Pipe som tar in ett nummer och returnerar detta med decimaltecken (default 2 decimaler, ex 12,00) 12 | komponentkartanCurrency
