Dokumentation för Tab button

Översikt
vgr-tab är en anpassad variant av knapp komponenten som ska kunna kopplas till en container. Den ska inte användas till navigering, utan är ett sätt att visa olika information eller funktionalitet inom ett bredare kontext. vgr-tab ska vara placerad direkt ovanpå containern (med 0 px mellanrum), men ska vara antingen centrerad med, eller vänster allignad med containern. Om containern är vgr-page-body ska tab vara i sidans vgr-page-header som längst upp på denna sidan.

Exempel

 En <vgr-tab-button> kan användas själv eller tillsammans med <vgr-tab-button-group> (rekommenderas). Man kan sätta storlek på tabbarna alt. låta tabbarna ta den storlek som namnet på tabben behöver. Man kan även sätta ett id på tabben för att kunna ha olika beteenden beroende på vilken knapp man tryckt på. Anger man inte något specifikt id som kommer en guid sättas. Tabbarna är en knapp i botten så kommer agera som en knapp

API

<vgr-tab-button>

Namn Beskrivning Exempel
ariaLabel: boolean Värde för aria-label. Förvalt värde är "[text] tab button",
där [text] är text content för <vgr-tab-button>.		 [ariaLabel]="En alternativ beskrivning."
active: boolean Talar om vilken knapp som är aktiverad. Om man använder tab-button-group
så sätts detta automatiskt, annars får man hantera det själv. Förvalt värde är false.		 [active]="true"
disabled: boolean Inaktivera tab button. Förvalt värde är false. [disabled]="true"
tabId: string Värde för id't på tabben. Förvalt värde är Guid.newGuid(). [tabId]="'Översikt'"
width: string Ett strängvärde för att sätta vilken bredd man vill ha på gruppen. Förvalt värde är
auto.		 [width]="'500px'"
focus() Metod för att flytta fokus till komponenten. this.myTabButton.focus()

<vgr-tab-button-group>

Namn Beskrivning Exempel
alignCenter: boolean Boolean för att centrera tabbarna. Förvalt värde är false. [alignCenter]="true"
width: string Ett strängvärde för att sätta vilken bredd man vill ha på gruppen. Förvalt värde är auto. [width]="'120px'"
focus() Metod för att flytta fokus till komponenten. this.myTabButtonGroup.focus()
