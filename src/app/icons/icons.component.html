Dokumentation för tillgängliga ikoner i komponentkartan

Det finns nu stöd för att använda Font Awsome (free). Genom att använda komponenten vgr-icon kan dessa ikoner användas https://fontawesome.com/icons?d=gallery&m=free. Per default så används temafärgen, detta kan överskridas med color attributet.
För implementation se dokumentation for vgr-icon.


Exempel på Font Awsome ikoner
[icon]="'pen'" [color]="'text'"
[icon]="'comment-dots'" [color]="'text'" [solid]="false"
[icon]="'check-circle'" [color]="'success'"
[icon]="'trash-alt'" [solid]="false" [color]="'text'"
[icon]="'plus'" [size]="'xs'"
[icon]="'minus'" [size]="'xs'"
[icon]="'check'"
[icon]="'user-alt'" [color]="'text'"
[icon]="'star'" [color]="'text'" [solid]="false"
[icon]="'star'" [color]="'text'"
[icon]="'question-circle'" [color]="'text'"
[icon]="'envelope'" [solid]="false"
[icon]="'print'"

Man kan fortfarande använda sig av de gamla gamla ikonerna nedan men dessa kommmar att ersättas av Font Awsome ikoner.

För att kunna använda sig utav de ikoner (svg-bilder) som finns i komponentkartan, så finns deras klassnamn representerade i tabellen nedanför.

Svg-bilderna är kopplade till en background-image för samtliga klasser i komponentkartans ikon-bibliotek.
