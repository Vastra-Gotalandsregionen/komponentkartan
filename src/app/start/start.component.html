Beskrivning

Detta är en komponentkarta av ersättningssystemen i Västra Götalandsregionen. Anledningen till att följa gemensamma riktlinjer vid navigation, kontroller och struktur är dels för att skapa en enhetlighet och dels kunna hantera flera system för samma inloggning. Innehållet i de olika ersättningssystemen är och kommer fortsättningsvis att vara systemspecifikt. Ersättningssystemen med gemensamt inloggning har en gemensam portal med en neutral färg som är skild från den röda, gröna och blå färgen. När en användare har behörighet för fler än ett system kommer de gemensamma delarna att visa den neutrala färgen.

Utecklingsmiljö för Angular + komponentkartan

  1. Installera Node https://nodejs.org/en/
    Verifiera att node och npm är installerat genom att köra node -v samt npm -v i en kommandoprompt (eller terminalen i VS Code).
  2. (Rekommenderat) Installera IDE Visual Studio Code. https://code.visualstudio.com/
  3. Installera angular-cli https://cli.angular.io/ globalt på din dator: npm i -g @angular/cli.
    Verifiera installationen genom att köra ng -v

Följ dessa steg om du sätter upp ett nytt projekt

  1. Skapa ett nytt projekt genom ng new ProjectName
  2. För att lägga till Komponentkartan i projektet, ställ dig i projektets root. Navigera till Komponentkartans GitHub-sida i din webbläsare och följ steg 1-2 på den sidan.
  3. Kör ng -s -o för att köra igång din applikation lokalt och öppna i ett webbläsarfönster.

Nu är det dags att börja bygga din applikation. Alla komponenter som finns i Komponentkartan finns dokumenterade under 'Komponenter' till vänster. Var noga med att sidstrukturen blir korrekt när du skapar nya sidor.

Följ dessa steg om du ska börja arbeta i ett befintligt projekt

  1. Hämta hem projektets repo från Komponentkartans GitHub-sida
  2. Ställ dig i projektets root och kör npm i för att installera alla node-paket som projektet behöver.
  3. Kör ng -s -o för att köra igång din applikation lokalt och öppna i ett webbläsarfönster.
